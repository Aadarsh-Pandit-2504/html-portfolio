Multiple Disease Prediction using Machine Learning

This project is a made to predict four diseases namely Diabetes,Heart Disease, Parkinson's Disease and Liver Disease. The Machine Learning Algorithms used here areSVM(Support Vector Machine),KNN(Kth Nearest Neighbour),LR(Logistic Regression) and RF(Random Forest). The accuracies obtained for various diseases are100%,90%,100% and 97% respectively.
